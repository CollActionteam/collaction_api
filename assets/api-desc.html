CollAction API

The CollAction API is a RESTful API that provides access to the data in CollAction. The API is available at https://api.collaction.org.

The API is versioned, and the current version is v1. This means when you access any of the API endpoints, you should append the version to the API url, like so: https://api.collaction.org/v1

The API is designed to be used by CollAction's Client Applications, such as the mobile app, and therefore the resources are designed to be easily consumed by these applications.
